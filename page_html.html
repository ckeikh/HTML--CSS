Le HyperText Markup Language, généralement
abrégé HTML ou dans sa dernière version HTML5,
est le langage de balisage conçu pour représenter
les pages web. Ce langage permet : d’écrire de l’hypertexte,
d’où son nom, de structurer sémantiquement la page,
de mettre en forme le contenu, de créer des formulaires de saisie,
